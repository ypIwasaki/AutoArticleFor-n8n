# AI Keyword Candidates

このフォルダには、AIキーワード抽出指示書に従って選定した検索キーワード候補を保存します。

- 作成者: AIまたは手動作業
- ファイル名: `YYYY-MM-DD.md`
- 指示書: 同日付の `../ai-extraction-instructions/YYYY-MM-DD.md`
- 根拠: 同日付の `../daily-digests/YYYY-MM-DD.md` と `../keyword-candidates/YYYY-MM-DD.md`

候補ごとにカテゴリ、確信度、既定キーワードへの追加可否、理由、根拠を記載します。このファイルの作成だけでは、n8nワークフローの既定キーワードは変更されません。
