# AI Summary Instructions

このフォルダには、取得済み記事の日次要約をAIへ依頼するための指示書を保存します。

- 作成者: n8nワークフロー
- ファイル名: `YYYY-MM-DD.md`
- 入力: 同日付の `../daily-digests/YYYY-MM-DD.md`
- 主な内容: 要約方針、対象キーワード、対象期間、出力形式、根拠として使う記事一覧

この指示書自体はAIを自動実行しません。指示書に従って作成した要約は、同日付の `../article-summaries/YYYY-MM-DD.md` に保存します。


## 本文確認

要約を作成するAIは、各記事URLを開いてリンク先の本文を確認してから要約します。RSSのタイトル・抜粋だけを言い換えた文章は要約として保存しません。本文を取得できない記事は、理由を添えて「本文未確認」と記録します。
