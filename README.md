# AutoArticleFor-n8n

This repository stores n8n workflow definitions and operating notes for a daily
keyword-based news and web-content summarization project.

## What is included

- `n8n/workflows/daily-keyword-news-summary.workflow.json`
  - Importable n8n starter workflow for daily keyword monitoring.
  - Runs on a daily schedule, manual trigger, or webhook request.
  - Reads Google News RSS and Hatena Bookmark RSS search results for configured keywords.
  - Produces a Markdown digest and an LLM prompt for higher-quality summarization.
- `n8n/workflows/auto-article-starter.workflow.json`
  - Earlier article-brief prototype. Prefer the daily keyword summary workflow
    for the current project direction.
- `docker-compose.yml`
  - Optional local n8n runtime using the official n8n container image.
- `.env.example`
  - Example local n8n environment variables. Do not commit real secrets.
- `docs/n8n-setup.md`
  - Setup, import, test, and workflow development notes.
- `docs/n8n-api-sync.md`
  - API-based sync instructions for updating an existing n8n workflow from this repository.
- `docs/ai-keyword-extraction.md`
  - Notes for the AI keyword extraction instruction Markdown generated by the workflow.
- `docs/ai-summary-instructions.md`
  - Notes for the AI article-summary instruction Markdown generated by the workflow.
- `content/`
  - Markdown archive generated by the workflow: daily digests, rule-based keyword candidates, AI keyword extraction prompts, and AI summary prompts.

## Quick start with npm

```bash
n8n
```

Open `http://localhost:5678`, then import:

```text
n8n/workflows/daily-keyword-news-summary.workflow.json
```

## Quick start with Docker

```bash
cp .env.example .env
docker compose up -d
```

Open `http://localhost:5678`.

## Workflow test payload

Use this JSON body with the webhook test URL:

```json
{
  "keywords": ["Vtuber", "にじさんじ"],
  "lookbackHours": 24,
  "maxArticles": 30,
  "locale": "ja-JP",
  "language": "ja",
  "country": "JP",
  "outputLanguage": "ja"
}
```

In n8n test mode, the webhook URL is usually:

```text
http://localhost:5678/webhook-test/daily-keyword-summary/request
```

After activating the workflow, use:

```text
http://localhost:5678/webhook/daily-keyword-summary/request
```

## Sync changes to n8n by API

After creating an n8n API key and setting `.env`, update the existing n8n
workflow from this repository:

```bash
python3 scripts/sync_workflow_to_n8n.py
```

See `docs/n8n-api-sync.md` for setup details.

## Markdown archive

When the workflow runs, it saves Markdown files under:

```text
content/daily-digests/
content/keyword-candidates/
content/ai-extraction-instructions/
content/ai-summary-instructions/
```

The daily digest file stores the collected articles and source-based digest. The
keyword candidate file extracts follow-up terms from the saved digest, with a
focus on VTuber agencies, groups, companies, and related terms. The instruction
files are ready-to-copy prompts for manually asking an AI chat to perform
semantic keyword extraction or article summarization. The workflow does not call
AI APIs automatically.

## Start n8n with project file access

To let n8n write generated Markdown into this project, start n8n with:

```bash
./scripts/start_n8n_with_file_access.sh
```

If n8n is already running, stop it first and start it again with this script.

## AI extraction instructions

The workflow does not call OpenAI or any other AI API. Instead, it writes a
Markdown instruction files under `content/ai-extraction-instructions/` and
`content/ai-summary-instructions/`. Copy either file into an AI chat when you
want semantic keyword extraction or article summarization manually.

See `docs/ai-keyword-extraction.md` for details.

## Weekly analysis

Each n8n run also writes `content/structured-records/YYYY-MM-DD.jsonl` for reusable analysis. Backfill existing Markdown digests and generate the latest weekly reports with:

```bash
python3 scripts/backfill_structured_records.py
python3 scripts/generate_analysis_reports.py
```

See `docs/analysis.md` for report contents and limitations.
